Traceability

Requirements use stable PRD-* IDs and open decisions use Q-* IDs. Tests, issues, ADRs and commits should reference these IDs.

HTML may condense Markdown but must preserve every normative rule, stable ID, command, path, status, risk, consequence and graph edge. Every graph needs a text/table equivalent. Run ./scripts/verify-doc-pairs.sh; semantic parity still requires review.
